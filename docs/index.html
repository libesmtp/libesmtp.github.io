libESMTP v 1.1.0¶

libESMTP is an SMTP client library which manages submission of electronic mail via a preconfigured Mail Transport Agent (MTA) such as Exim or Postfix. It implements many SMTP extensions including TLS for security and PIPELINING for high performance.

The current version of libESMTP is 1.1.0.

Earlier versions are not recommended.

This is the official libESMTP repository and documentation. It is maintained by Brian Stafford, the original libESMTP author.
